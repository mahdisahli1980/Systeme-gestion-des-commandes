# Système de gestion de commandes — Python

Projet Python en mode console pour gérer des commandes clients.

## Fonctions
- Ajouter un produit au panier
- Supprimer un produit
- Afficher la commande
- Valider la commande
- Sauvegarder l’historique dans un fichier texte

## Données
Le programme lit un fichier CSV de produits Amazon :
produits_amazon (1).csv

## Lancement

Installer pandas :
pip install pandas

Exécuter :
python mon_projet.py
